Snapshot basis:

4. Treatment card section

7. Message section

8. Info/contact section

9. Footer

Key responsive behavior observed:
- Separate PC/mobile header structures are present in the DOM.
- Core sections remain vertically stacked on desktop, tablet, and mobile.
- The treatment section collapses from multi-column cards to narrower stacked cards.
- The location section shifts from split image/map layout toward a single-column flow on smaller widths.
- Footer navigation compresses into a tighter stacked layout on smaller widths.
